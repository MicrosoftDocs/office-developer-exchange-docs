# UniqueBody

The **UniqueBody** element represents an HTML fragment or plain text which represents the unique body of this conversation. 
  
```XML
<UniqueBody BodyType=""/>
```

 **BodyType**
## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
### Attributes

|**Attribute**|**Description**|
|:-----|:-----|
|**BodyType** <br/> |Describes how the item body is stored in the item.  <br/> |
   
#### BodyType Attribute

|**Value**|**Description**|
|:-----|:-----|
|**HTML** <br/> |Converts all bodies to HTML.  <br/> |
|**Text** <br/> |Converts all bodies to plain text.  <br/> |
