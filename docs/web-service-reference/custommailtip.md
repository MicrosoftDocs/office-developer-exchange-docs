# CustomMailTip

The **CustomMailTip** element represents a customized mail tip message. 
  
```XML
<CustomMailTip/>
```

 **string**
## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
#### Attributes

None.
  
#### Child elements

None.
  
#### Parent elements

|**Element**|**Description**|
|:-----|:-----|
|[MailTips](mailtips.md) <br/> |Represents values for the various types of mail tips.  <br/> |
   
## Text value

A text value that represents a string is required if this element is used.
  
## Remarks

The schema that describes this element is located in the IIS virtual directory that hosts Exchange Web Services.
