# AlternateMailboxes (SOAP)

The **AlternateMailboxes** element represents a collection of alternate mailboxes. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
