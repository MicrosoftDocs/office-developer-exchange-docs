# LocationUrl (SOAP)

The **LocationUrl** element represents the URL of a document sharing location. 
  
```XML
<LocationUrl />
```

 **string**
## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
### Attributes

None.
  
### Child elements

None.
  
### Parent elements

|**Element**|**Description**|
|:-----|:-----|
|[DocumentSharingLocation (SOAP)](documentsharinglocation-soap.md) <br/> |Represents location and metadata information for a document sharing location.  <br/> |
   
## Text value

The text value of the **LocationUrl** element is the URL of a document sharing location.
