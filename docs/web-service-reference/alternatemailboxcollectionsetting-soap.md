# AlternateMailboxCollectionSetting (SOAP)

The **AlternateMailboxCollectionSetting** element contains a collection of alternate mailbox settings. 
  
```XML
<AlternateMailboxCollectionSetting>
   <AlternateMailboxes/>
</AlternateMailboxCollectionSetting>
```
