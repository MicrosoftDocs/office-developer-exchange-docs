# State

The **State** element represents the state of residence for a contact item. 
  
```xml
<State/>
```

**string**

## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
### Attributes

None.
  
### Child elements

None.
  
### Parent elements

|**Element**|**Description**|
|:-----|:-----|
|[Entry (PhysicalAddress)](entry-physicaladdress.md) <br/> |Describes a single physical address for a contact item.  <br/> |
   
## Text value

A string value that represents the name of a state is required if this element is used.
  
## Remarks

The schema that describes this element is located in the EWS virtual directory of the computer that is running MicrosoftExchange Server 2007 that has the Client Access server role installed.
