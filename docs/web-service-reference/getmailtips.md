# GetMailTips

The **GetMailTips** element represents the recipients and types of mail tips to retrieve. 
  
```XML
<GetMailTips>
   <SendingAs/>
   <Recipients/>
   <MailTipsRequested/>
</GetMailTips>
```

 **GetMailTipsType**
## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
### Attributes

None.
  
### Child elements

|**Element**|**Description**|
|:-----|:-----|
|[SendingAs](sendingas.md) <br/> |Contains an e-mail address that a user is trying to send as.  <br/> |
|[Recipients (ArrayOfRecipientsType)](recipients-arrayofrecipientstype.md) <br/> |Contains a list of recipients to check for mail tips.  <br/> |
|[MailTipsRequested](mailtipsrequested.md) <br/> |Contains the types of mail tips requested from the service.  <br/> |
   
### Parent elements

None.
  
## Text value

None.
  
## Remarks

The schema that describes this element is located in the IIS virtual directory that hosts Exchange Web Services.
