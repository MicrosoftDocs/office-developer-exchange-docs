# IsDefault (SOAP)

The **IsDefault** element indicates whether a document sharing location is the user's default sharing location. 
  
```XML
<IsDefault /> 
```

 **Boolean**
## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
### Attributes

None.
  
### Child elements

None.
  
### Parent elements

|**Element**|**Description**|
|:-----|:-----|
|[DocumentSharingLocation (SOAP)](documentsharinglocation-soap.md) <br/> |Represents location and metadata information for a document sharing location.  <br/> |
   
## Text value

The Boolean value of the **IsDefault** element indicates whether the sharing location is the user's default sharing location. 
  
## Element information

|**Name**|**Value**|
|:-----|:-----|
|Namespace  <br/> |https://schemas.microsoft.com/exchange/2010/Autodiscover  <br/> |
|Schema Name  <br/> |Autodiscover schema  <br/> |
|Validation File  <br/> |Messages.xsd  <br/> |
|Can be Empty  <br/> |True  <br/> |
