# Type (SOAP)

The **Type** element represents the alternate mailbox type. 
  
```XML
<Type/>
```

 **string**
## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
### Attributes

None.
  
### Child elements

None.
  
### Parent elements

|**Element**|**Description**|
|:-----|:-----|
|[AlternateMailbox (SOAP)](alternatemailbox-soap.md) <br/> |Represents an alternate mailbox.  <br/> |
   
## Text value

The text value of the **Type** element is the alternate mailbox type. 
  
## Element information

|**Name**|**Value**|
|:-----|:-----|

|Validation File  <br/> |Messages.xsd  <br/> |
|Can be Empty  <br/> |True  <br/> |
