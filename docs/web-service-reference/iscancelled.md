# IsCancelled

The **IsCancelled** element indicates whether an appointment or meeting has been canceled. 
  
```xml
<IsCancelled/>
```

 **Boolean**
## Attributes and elements

The following sections describe attributes, child elements, and parent elements.
  
### Attributes

None.
  
### Child elements

None.
  
### Parent elements

|**Element**|**Description**|
|:-----|:-----|
|[CalendarItem](calendaritem.md) <br/> |Represents an Exchange calendar item.  <br/> |
|[MeetingRequest](meetingrequest.md) <br/> |Represents a meeting request in the Exchange store.  <br/> |
   
## Text value

A text value that represents a Boolean value is required if this element is included. A value of **true** indicates that the calendar item has been canceled. A value of **false** indicates that a calendar item has not been canceled. 
  
## Remarks

The schema that describes this element is located in the EWS virtual directory of the computer that is running MicrosoftExchange Server 2007 that has the Client Access server role installed.
